# Sentiment Analysis Dashboard

An interactive dashboard for analyzing social media sentiment. Users can upload text files and visualize positive, negative, and neutral sentiment trends.

## Features
- File upload support
- Real-time sentiment visualization
- Interactive charts and graphs

## Tech Stack
- Python
- Streamlit
- TextBlob
- Pandas
- Matplotlib
